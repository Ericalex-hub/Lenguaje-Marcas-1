div.item{THIS IS ITEM$}*8

Lo que hace esto es crear un div de clase item que contiene la frase "THIS IS ITEM",junto con un "$"que son los numero que se van poniendo a lado y hace todo esto ocho veces.

THIS IS ITEM1
THIS IS ITEM2
THIS IS ITEM3
THIS IS ITEM4
THIS IS ITEM5
THIS IS ITEM6
THIS IS ITEM7
THIS IS ITEM8

CTRL + F2

Lo que hace esto es que seleciona todas la palabras que son iguales a la selecionada, esto se usa sobre todo cuando tienes una clase en muchos div y quieros cambiar todos a otra.

ALT + ^ o v

Lo que esto hace es que al presionar ALT y la flecha hacia arriba o hacia abajo, mueve la linea o bloque de código seleccionado hacia arriba o abajo.

CSS: df+jcc+aic+g20px+p40px

Lo que hace esto, en CSS es cuanto está dentro de una clase a editar, crea un display: flex, un justify-content: center, un align-item: center, un gap: 20px y un padding: 40px. Para poder saber utilizarlo, solo se hade de escribir la/s primera/s letras de cada propiedad yluego la primera letra o valor que tiene esa propiedad.
